# Incremental Compile 
Incremental Compile (증분 컴파일)이란 프로그램의 수정된 부분만 다시 컴파일하는 기술을 의미한다.

# Incremental Build
Incremental Build (증분 빌드)란 이미 빌드된 파일을 모두 다시 빌드하는 것이 아닌, 바뀐 파일들만 빌드하는 것을 뜻한다.
