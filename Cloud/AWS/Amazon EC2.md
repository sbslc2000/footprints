# Amazon EC2
Amazon Elastic Compute Cloud(Amazon EC2) 란 Amazon에서 제공하는 클라우드 확장식 컴퓨팅이다. EC2를 사용하면 하드웨어에 선투자 할 필요 없이 더 빠르게 애플리케이션을 개발하고 배포할 수 있다. EC2에는 원하는 만큼 가상 서버를 구성할 수 있고, 네트워크 구성과 스토리지 관리까지 도와준다. EC2는 요구사항이나 갑작스러운 트래픽 증대 등 변동 사항에 따라 신속하게 규모를 확장하거나 축소할 수 있어 서버 트래픽 예측 필요성을 줄여준다.

## Bin Packing

Bin Packing이란 EC2 인스턴스 위에서 여러 컨테이너를 실행할 때, 각 컨테이너의 CPU와 메모리 요구량이 다르기 때문에 어떤 컨테이너를 어떤 인스턴스에 배치할지 계산해야 하는 문제다. 짐을 최대한 효율적으로 박스에 담는 것에 비유할 수 있다.

ECS on EC2 환경에서는 컨테이너 오케스트레이터가 bin packing을 자동으로 처리한다. 그러나 인스턴스 수와 타입 선택, 인스턴스 간 자원 낭비 최소화 등은 여전히 운영자가 고민해야 한다. [[AWS Fargate]]를 사용하면 이 문제 자체가 사라진다.
