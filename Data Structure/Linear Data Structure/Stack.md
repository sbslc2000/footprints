# Stack
스택*Stack*이란, **후입 선출**(LIFO, Last-In-First-Out) 방식으로 동작하는 선형 자료구조이다. 가장 최근에 추가된 데이터가 가장 먼저 제거되는 구조를 가진다.

## 주요 연산
* push : 데이터를 스택에 추가한다.
* pop : 데이터를 스택에서 제거하고 반환한다.
* peek, top : 가장 최근에 저장한 데이터를 조회한다.

## 주요 사용처
* 함수 호출 관리 
	* 이는 call stack 이라고도 부른다.
* 괄호 검사
* 웹 브라우저의 뒤로가기 기능
