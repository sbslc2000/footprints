# Factory Method Pattern
## Intent
Factory Method Pattern은 **상속**을 사용하여 구체적인 객체 생성 로직을 서브클래스에서 구현하도록 한다.

Factory Method Pattern은 어떠한 클래스가 작성되는 시점에 해당 클래스에서 특정한 인스턴스를 생성하는 것을 미룬다. 이후 미뤘던 과정은 서브클래스에 의해서 구체적인 로직으로 작성된다.
