# Spring Data COMMON

스프링에서 다양한 저장 방식을 지원하는 공통적인 역할을 하는 모듈이며, 이곳에 Repository를 빈으로 등록하거나 쿼리 메서드를 만드는 매커니즘이 들어있다. COMMON에 대해 잘 이해하고 있으면 다른 저장 방식을 사용하는 기술을 사용하더라도 빠르게 익힐 수 있다.
