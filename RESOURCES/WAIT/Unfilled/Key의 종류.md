# Key의 종류

### Super Key
> a subset of attributes of relation schema that satisfies Uniqueness

> Every relation has at least one default superkey

유일성의 특성을 만족하는 속성 또는 속성들의 집합

### Key
Candidate Key라고도 불린다.
Attribute ( or attributes combination) that can uniquely identify every single tuples in a relation schema 튜플을 찾거나 순서대로 정렬할 때 다른 튜플들과 구별할 수 있는 기준이 되는 attribute
Key is minimal superkey for given attributes combination Minimum
유일성과 최소성을 만족하는 속성 또는 속성들의 집합.
