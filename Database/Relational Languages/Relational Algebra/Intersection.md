# Intersection
$$ R \cap S $$ 합병이 가능한 두 릴레이션 R과 S의 교집합은 위와 같이 표현한다. $R \cap S$ 는 릴레이션 R과 S에 속하는 모든 투플로 결과 릴레이션을 구성한다.

## Example

Relation R

| ID  | name  |
| --- | ----- |
| 100 | Pete  |
| 200 | John  |
| 300 | James |
Relation S

| ID  | name |
| --- | ---- |
| 100 | Pete |
| 400 | Lee  |
$R \cap S$

| ID  | name  |
| --- | ----- |
| 100 | Pete  |
